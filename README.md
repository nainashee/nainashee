## 🏗️ What I'm Building

### 🏏 Live Products

**[Howzat — playhowzat.com](https://playhowzat.com)** · [`cricket-zone`](https://github.com/nainashee/cricket-zone)
Daily cricket player guessing game. Identify legends from their silhouette.
Built serverless on AWS — Lambda, DynamoDB, Cognito, CloudFront, API Gateway.
Full CI/CD via GitHub Actions. PWA with offline support.

### ☁️ Cloud Infrastructure

**[SandCastle](https://github.com/nainashee/sandcastle)**
A castle in the cloud — personal AWS dev environment built with Terraform.
Secured with SSM Session Manager, cost-optimized with auto-stop scheduling.
No SSH keys, no IAM users, no surprise bills.

**[LaunchPad](https://github.com/nainashee/launchpad)** *(in progress)*
AI-powered job search automation platform on AWS. 16-week build.
DynamoDB + Lambda + API Gateway, deployed at jobs.naindegital.com.

### 📚 Hands-on Labs & Learning

**[AWS Cloud Labs](https://github.com/nainashee/aws-cloud-labs)**
Structured 24-week AWS lab program covering VPC, IAM, EC2, S3, CloudWatch,
Terraform, GitHub Actions, Docker, and boto3.

**[EC2 Health Monitor](https://github.com/nainashee/Phase-1-Capstone-EC2-Health-Monitor-with-S3-Logging-and-CloudWatch-Alerts)**
Automated EC2 health monitoring — reads CPU metrics from CloudWatch,
writes JSON logs to S3, triggers SNS email alerts on threshold breach.
Built with Python/boto3 and IAM role-based access.
